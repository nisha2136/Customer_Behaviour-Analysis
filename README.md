# 📊 Customer Behavior Dashboard

## 📌 Overview  
This project analyzes customer behavior using Python, MySQL, and Power BI. The goal is to understand purchasing patterns and present insights through an interactive dashboard.

---

## 🛠️ Tools Used  
- Python (EDA & Data Cleaning)  
- MySQL (SQL Queries)  
- Power BI (Dashboard)  

---

## 🔄 Workflow  
- Loaded and cleaned dataset using Python  
- Performed Exploratory Data Analysis (EDA)  
- Ran SQL queries in MySQL for insights  
- Built an interactive Power BI dashboard  

---

## 📁 Dataset  
The dataset contains customer-related information such as:
- Customer demographics (Age, Gender)  
- Product categories (Clothing, Accessories, Footwear, Outerwear)  
- Purchase amount and review ratings  
- Subscription status and shipping type  

---

## 📊 Key Insights  
- Clothing category generates the highest sales and revenue  
- Young customers contribute the most purchases  
- Average customer rating is **3.75**  
- Average purchase amount is **$59.87**  
- Majority of customers are non-subscribers  

---

## 📂 Project Files  
- `customer_shopping_behavior.csv` – Raw dataset
- `Customer_Behavior.ipynb` – Data Cleaning & EDA
- `customer_behavior.pbix` – Power BI dashboard  

---

## 🚀 Conclusion  
This project demonstrates an end-to-end data analytics workflow, from data cleaning and analysis to visualization. It highlights how data can be used to understand customer behavior and support business decision-making.
